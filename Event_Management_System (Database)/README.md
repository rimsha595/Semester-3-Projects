# Event Management System (Database)

## Description
This project is an Event Management System developed using SQL and database management concepts. It is designed to manage events, participants, and scheduling efficiently.

## Features
- Add new events
- Manage participants and registrations
- Store event details in database
- Perform SQL queries for data retrieval
- Maintain organized event records

## Technologies Used
- SQL
- Database Management System (DBMS)

## Database Design
- Events Table
- Participants Table

## Concepts Covered
- Relational database design
- SQL queries (SELECT, INSERT, UPDATE, DELETE)
- Table relationships

## Purpose
To understand database systems and how real-world event management systems work using SQL.
