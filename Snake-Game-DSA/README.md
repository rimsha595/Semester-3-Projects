# Snake Game using Data Structures and Algorithm

## Description
This project is a Snake Game developed using Data Structures and Algorithms (DSA), specifically the Linked List concept. The game also integrates a database to store and manage game-related data.

## Features
- Snake movement using Linked List logic
- Food generation and score tracking
- Collision detection (wall and self-collision)
- Game state management
- Database integration for storing scores/data

## Technologies Used
- Programming Language: C++ / Python (add yours)
- Data Structures: Linked List

## Concepts Covered
- Linked List implementation
- Dynamic memory handling
- Game logic design
